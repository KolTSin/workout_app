# Beginner-safe Templates

Design goals:

- Easy to execute
- Low injury risk
- Minimal equipment assumptions
- Consistent movement patterns
- Progression-friendly

## Upper (Template A)

**Warmup**

- 5 min easy bike/row
- 10 band pull-aparts + 10 scap push-ups (optional)

**Exercises**

1. **Dumbbell Bench Press** (PUSH_H) — 3×6–10 @RPE 7–8, rest 120s
   - Subs: Machine Chest Press, Barbell Bench, Push-ups
2. **Chest-Supported Row** (PULL_H) — 3×8–12 @RPE 7–8, rest 90s
   - Subs: Seated Cable Row, One-arm DB Row
3. **Seated DB Shoulder Press** (PUSH_V) — 2×8–12 @RPE 7–8, rest 90s
   - Subs: Machine Shoulder Press
4. **Lat Pulldown** (PULL_V) — 2×8–12 @RPE 7–8, rest 90s
   - Subs: Assisted Pull-ups, Cable Pullover
5. **Cable Curl** — 2×10–15 @RPE 8, rest 60s
6. **Triceps Pressdown** — 2×10–15 @RPE 8, rest 60s
7. **Dead Bug / Plank** — 3 sets

## Lower (Template A)

**Warmup**

- 5 min incline walk or bike
- 10 bodyweight squats + 10 hip hinges

**Exercises**

1. **Leg Press** (SQUAT) — 3×8–12 @RPE 7–8, rest 120s
   - Subs: Goblet Squat, Hack Squat, Smith Squat
2. **Dumbbell Romanian Deadlift** (HINGE) — 3×6–10 @RPE 7–8, rest 120s
   - Subs: Machine/Smith RDL, Back Extensions
3. **Split Squat** (UNILATERAL) — 2×8–12 each leg @RPE 7–8, rest 90s
   - Subs: Walking Lunges, Step-ups
4. **Seated/lying Leg Curl** — 2×10–15 @RPE 8, rest 60–90s
5. **Calf Raise** — 2×10–15 @RPE 8, rest 60s
6. **Side Plank** — 2–3 sets

## Full Body (Template A)

**Warmup**

- 5 min row/bike
- 1–2 ramp-up sets for first exercise

**Exercises**

1. **Goblet Squat** (SQUAT) — 3×6–10 @RPE 7–8, rest 120s
   - Subs: Leg Press, Hack Squat
2. **DB Bench Press** (PUSH_H) — 3×6–10 @RPE 7–8, rest 120s
   - Subs: Machine Press
3. **Lat Pulldown** (PULL_V) — 3×8–12 @RPE 7–8, rest 90s
   - Subs: Assisted Pull-ups
4. **DB RDL OR Back Extension** (HINGE) — 2×8–12 @RPE 7–8, rest 120s
5. **Farmer Carry** (optional) — 4×20–40m (controlled)
6. **Core** — 2–3 sets

## Cardio (two types)

**Cardio A — Zone 2**

- 25–40 min steady state
- “Can talk in full sentences” pace
- Equipment: bike / treadmill incline walk / elliptical / rower

**Cardio B — Intervals (beginner-safe)**

- 5–8 rounds:
  - 60s hard (RPE 8)
  - 90s easy (RPE 3)
- 5 min warmup + 5 min cooldown
